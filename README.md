# TROCAPP
Um App para troca e doações

## Dependências
- NodeJS >= 8
- Yarn

## Instalar
git clone https://github.com/belemzeiros/trocapp
```
yarn install
```

## Compilar
yarn build
yarn start

## Testes
yarn test

## Boas práticas
yarn eslint:dev
yarn eslint:dev --fix
